# Mochi — Kawaii shop

Složka pro nový e-shop **Mochi** (kawaii styl).

## Kam vložit podklady

| Složka | Co sem dát |
|--------|------------|
| `docs/reference/` | Návrh stránky — HTML, screenshot, PDF, Canva export, popis designu |
| `assets/` | Loga, obrázky produktů, ikony, fonty |

Po nahrání souborů napiš v Cursoru zprávu — stránku postavíme podle podkladů a nasadíme na **GitHub** + **Vercel**.

## Plánovaný stack

- Next.js (stejně jako Neurea)
- Vercel hosting
- Repozitář: `mochi-shop` na GitHubu (vytvoříme při nasazení)
